# budgetTracker
the budgetTracker webapp is intended to afford the user the ability to log various financial transactions, deposit or withdrawl, and track how any specific transactions impacts the overall balance.

## Technology
* JS
* HTML
* CSS
* MongoDB
* Heroku
* Mongo Atlas
* IndexedDB
* Node
